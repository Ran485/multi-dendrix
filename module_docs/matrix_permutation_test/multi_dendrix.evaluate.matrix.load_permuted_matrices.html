multi_dendrix.evaluate.matrix.load_permuted_matrices¶

multi_dendrix.evaluate.matrix.load_permuted_matrices(input_dir)[source]¶

Loads all files as mutation data in a directory of permuted mutation data.

Parameters:input_dir (string) – location of mutation data directory.
Returns:a list of mutation data tuples (for more information on the mutation data format used internally by Multi-Dendrix, see multi_dendrix.load_mutation_data()).

See also: matrix_permutation_test().
