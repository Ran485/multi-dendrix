multi_dendrix.subtypes.keep_significant¶

multi_dendrix.subtypes.keep_significant(gene2specificity, threshold)¶

Removes all associations in the input dictionary that are not significant below a given threshold.

Parameters:
  • gene2specificity (dictionary) – maps each gene to a dictionary of subtypes, where each subtype is mapped to the result of the statistical test (see output of subtype_analysis()).
  • threshold (float) – Significance threshold.

An example is given in the documentation of subtype_analysis().

See also: subtype_analysis(), subtype_specificity().
