multi_dendrix.subtypes.subtype_specificity¶

multi_dendrix.subtypes.subtype_specificity(gene, patient2ty, ty2numpatients, mutation2patients)¶

Performs a statistical test on a gene for each given (sub)type.

Test:
For a given (sub)type T and gene g we calculate the statistical association of mutations in g to (sub)type T as follows:
  1. Construct the following contingency table:
  1. Perform Fisher’s exact test on the above contingency table.
  2. Bonferonni-correct the p-value.
Parameters:
  • gene (string) – gene name.
  • patient2ty (dictionary) – mapping of samples to their respective (sub)types (see load_patient2ty_file()).
  • ty2numpatients (dictionary) – mapping of each (sub)type to the number of patients of that subtype.
  • mutation2patients (dictionary) – mapping of genes to the patients in which they are mutated (see multi_dendrix.load_mutation_data() for details).
Returns:
A dictionary that maps each (sub)type to the following tuple:
(x11, x10, x01, x00, p-value, Bonferonni-corrected p-value).
Examples:
A view of the data:
>>> pnt2ty = {"TCGA-01" : "Luminal A", "TCGA-02" : "Luminal A", "TCGA-03" : "Luminal B"}
>>> ty2numpatients = {"Luminal A" : 2, "Luminal B" : 1}
>>> mutation2patients = {"G1" : ["TCGA-01", "TCGA-02", "TCGA-03"],
                         "G2" : ["TCGA-02"], "G3" : ["TCGA-01", "TCGA-02"]}
Simple example:
>>> subtype_specificity("G3", pnt2ty, ty2numpatients, mutation2patients)
{'Luminal A': [2, 0, 0, 1, 0.33, 0.66], 'Luminal B': [0, 1, 2, 0, 1.0, 1]}

See also: subtype_analysis(), ty_contingency_table().
